# Humans over AI

An interactive investigative journalism piece about AI-assisted military targeting, the Minab school strike, and the systematic dismantling of civilian harm oversight.

**[→ View live site](https://sophieverse.github.io/humans-over-ai)**

---

Built with HTML, CSS, and vanilla JavaScript. Hosted on GitHub Pages.

Features a terminal-style intro sequence, scroll-triggered animations, animated stat counters, and a full chronological timeline.

*Sources: New York Times · Brennan Center for Justice · WNYC On the Media · Breaking Defense*
